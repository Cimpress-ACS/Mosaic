The PLC base concept is a standardized approach how to structure and name a hierarchical state machine tree. It's a specification which describes naming conventions etc.<br /></p><p><br /></p><p>The .NET PlcCommunication library supports this PLC base concept specification by providing standardized enums, classes and importers.<br /></p><p><br /></p><p>The centralized main component is the Controller. A controller represents a piece of hardware and is pluggable into a hierarchical controller-tree. It accepts commands, is configurable, has parameters and alarms.<br /></p>
