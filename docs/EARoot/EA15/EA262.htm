This diagram shows the simulation concept of LineControl. The goal is to be able to run LineControl without a real hardware and event to run automated deterministic integration tests based on this simulator.<br /></p><p>The idea it to abstract the hardware with simple mocks. The ModuleSimulator class is a hardware mock and implements an item queue with behavior. It follows the modular approach but is completely decoupled from LineControl modules, those stay unchanged and should not recognize any difference when running LineControl in simulated mode or with real hardware.<br /></p>
