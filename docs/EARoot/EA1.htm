Shows an overview of the Deployments and the Modules within a Deployment.<br /></p><p><br /></p><p>Modules communicates via WCF.<br /></p><p><br /></p><p>The LineControlService is the main application and comes without UI. It communicates with PLC and handles the business logic.<br /></p><p><br /></p><p>The RemoteMaintenance application provides an UI and could be deployed locally on the machine, on a developer PC for remote debugging and a Windows Tablet. So we have here multiple instances.<br /></p><p><br /></p><p>The RemoteOperator application provides an UI for operators and could be deployed multiple times as well for each loading station. But only one loading station is able to do quality checks (configurable).<br /></p><p><br /></p><p>In the end we have three different installer: For LineControlService, RemoteMaintenance and RemoteOperator.<br /></p>
